GeoExt Styler README
====================

Installation
------------

To install the GeoExt styler simply unzip 'styler-*.zip' into the 'www' 
directory of the GeoServer installation. No restart of GeoServer is necessary.

Usage
-----

Once installed the styler is available at: 

   'http://<host>:<port>/geoserver/www/styler/index.html 

See http://geoserver.org/display/GEOS/GeoExt+Styler for more information.
